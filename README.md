# Jerickson Abreu - CV Landing Page

This is a personal landing page built with React, Vite, and Tailwind CSS, based on the provided CV.

## Project Structure

- `src/components`: UI Components (Hero, Experience, Skills, etc.)
- `src/data/profile.js`: Centralized data file containing all CV information.
- `src/App.jsx`: Main layout.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run locally:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Customization

To update the content, edit `src/data/profile.js`.
To change styles, modify `src/index.css` or `tailwind.config.js`.
